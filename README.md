# 住環境見取り図アプリ（mitorizu-app）

ケアマネジメント・訪問系の支援職が、訪問先の住宅環境の見取り図（間取り図）を、ブラウザだけでサクッと描ける無料ツールです。

**公開URL：** https://kat-t00.github.io/mitorizu-app/

インストール・会員登録・ダウンロードは一切不要。リンクを開けばすぐ使えます。

---

## このアプリの位置づけ

住宅改修の申請書類のような精密な採寸図面ではありません。**「動線」「段差」「手すりの位置」など、アセスメントに必要な情報をざっくり分かりやすく残す**ための簡易スケッチツールです。正式な採寸・申請図面は福祉用具事業者等のテンプレートをご利用ください。

---

## 主な機能

### 部屋・家具設備の配置
- 部屋（四角）をドラッグで配置・移動、四隅どこからでもリサイズ（部屋同士は辺が自動で揃う吸着あり）
- 開き戸・引き戸・両開き戸・親子扉・折れ戸・窓・開口部（勝手口など扉のない出入口）は、部屋の壁際まで近づけると自動で壁に開口部ができてフィットする
- 便器・浴槽・洗面台・キッチン・ベッド・テーブル・車椅子・ポータブルトイレ・手すり（縦/横/L型）・段差解消・スロープ・階段・方位記号・動線矢印など、アセスメントに必要な設備アイコンを一通り収録
- 一覧にないもの（押入・仏壇など）は「✏️ 自由入力」から形と名前を決めて追加可能
- 壁付けでない家具は、部屋の壁際にぴったり吸着させることも、あえて隙間を空けて配置することもできる
- 家具・階段は四隅のハンドルでリサイズ、自由な角度への回転、左右反転（丁番側の変更）に対応

### 出力・保存
- 画像（PNG）・PDF出力（透明背景、部屋の床の色だけ透明にする、といったオプションにも対応）
- PDF出力には図記号の凡例・作成者名も同梱可能
- 複数ケースをブラウザに自動保存、ファイルへの書き出し・読み込みにも対応
- 元に戻す／やり直す（Undo/Redo）

---

## 動作環境

- 対応ブラウザ：Chrome / Edge / Safari などの最新ブラウザ
- サーバー不要、データは基本的にお使いの端末（ブラウザ）内に保存されます
- 個人情報を含むデータを外部に送信する機能はありません（完全にブラウザ内で動作）
- データはブラウザ（端末）ごとの保存です。複数端末で共有したい場合は「💾 データ保存」でファイルに書き出し、手動で受け渡してください

---

## 使い方

1. 公開URLを開く
2. 「🆕 はじめから作成」で対象者名・案件名を入力してスタート
3. 「➕ 部屋を追加」で部屋を配置し、左側の「🪑 家具・設備を追加」から必要なものを配置
4. ドア・窓は壁際までドラッグすると自動でフィット。家具はクリックで編集・複製・削除ができる
5. 「🖼 画像出力」または「📄 PDF出力」で書き出し

画面右上の「❓ 使い方ガイド」に、操作方法の一覧をまとめています。

---

## 既知の制限

- 精密な採寸・申請図面としての利用は想定していません（アセスメント用の簡易スケッチが目的です）
- タッチ操作（タブレット等）での細かいハンドル操作は、マウス操作に比べて掴みにくい場合があります
